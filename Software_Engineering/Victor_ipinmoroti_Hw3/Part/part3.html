(a) Test first development is when one test the software program after every change has been made.
(b) Test driven development is when the test is actually a program and it is written before the code even specify the requirement of the program. Every time a functionality is added the old test and new test runs again to make sure there are no error.
(c) A sprint is is a development cycle that spans 2 to 4 weeks before the new version of the software is delivered to the customer and feedback is received for the next cycle.
(d) A scrum is really just a daily meeting to review progress made in the sprint.
